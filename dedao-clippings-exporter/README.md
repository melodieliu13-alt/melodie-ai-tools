# 得到 Clippings 导出器

这是独立于 WorkBuddy 的新扩展，用来把得到文章直接保存成 Obsidian Clippings 风格 Markdown。

加载目录：

```text
/Users/melodie2026/Documents/MelodieOS 2/dedao-clippings-exporter/extension
```

刷新得到页面后，右上角会出现三个按钮：

```text
提取全部文章URL
导出MD到Clippings
只导出当前文章
```

第一次导出时，Chrome 会要求选择文件夹。请选择：

```text
/Users/melodie2026/Documents/MelodieOS/Clippings
```

建议先点「只导出当前文章」测试格式，再跑整门课。
